Overview

The MetaLens API allows developers to extract EXIF metadata, GPS coordinates, camera details, and image properties programmatically.

Authentication

Include your API key in the request header:

X-API-Key: YOUR_API_KEY

Endpoint

POST https://api.metalens.dev/v1/extract

Send an image file as multipart/form-data.

Example Request


curl -X POST https://api.metalens.dev/v1/extract \
  -H "X-API-Key: YOUR_API_KEY" \
  -F "image=@photo.jpg"

Response Format


{
  "width": 4032,
  "height": 3024,
  "camera": "iPhone 14 Pro",
  "iso": 125,
  "aperture": 1.8,
  "gps": {
    "lat": 37.7749,
    "lng": -122.4194
  }
}

Rate Limits

  • Free Tier: 100 requests/day
  • Pro Tier: 10,000 requests/day
  • Enterprise: Unlimited
